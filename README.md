# Project Overview

## Noxis

Noxis is a cutting-edge software project designed to bring innovative solutions to modern computing challenges. The project is meticulously crafted to ensure high performance and compatibility across various systems.

### Language Composition
- **C++**: 86.6%  
- **PowerShell**: 7.1%  
- **Assembly**: 4.8%  
- **Makefile**: 1.3%  
- **Linker Script**: 0.2%  

This diverse array of languages contributes to Noxis's robustness and flexibility, enabling both high-level operations and low-level system interactions.

### Features
- Extensive use of modern C++ to leverage the latest language features.
- Scripting capabilities through PowerShell to automate workflows and improve user experience.
- Efficient system-level interactions via Assembly.

### Getting Started
To get started with Noxis, clone the repository and build the project using the provided Makefile. Ensure all dependencies are installed and properly configured.

For contributing, please refer to the CONTRIBUTING.md file for guidelines.
